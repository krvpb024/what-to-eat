# what-to-eat 吃什麼
這是一個使用[Vue.js](https://github.com/vuejs/vue)製作的PWA，當一群人無法決定要吃什麼的時候，就交給機率來決定。

## 功能介紹
分為三個層級：群組、類別、選擇。比如說現在在公司（群組），要決定午餐（類別）吃什麼，有OO便當、XX水餃、OX牛肉麵（選擇）。
